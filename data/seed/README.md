# Seed Data

This directory stores deterministic seed manifests used for local demos and CI smoke tests.

Primary seed corpus:

- `examples/enterprise_docs/` for raw heterogeneous docs
- `examples/sample_okf_bundle/` for precompiled OKF references
- `examples/eval/agent_benchmark.json` for agent QA benchmark

Use this manifest-driven seed setup to ensure reproducible local and CI runs.
