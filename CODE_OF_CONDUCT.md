# Code of Conduct

## Our Commitment

We are committed to creating an open, respectful, and professional environment for everyone participating in this project.

## Expected Behavior

- Be respectful and constructive.
- Assume positive intent and discuss ideas, not people.
- Provide actionable technical feedback.
- Keep communication focused on collaboration and project outcomes.

## Unacceptable Behavior

- Harassment, discrimination, or abusive language.
- Personal attacks or intimidation.
- Deliberate disruption of project workflows.

## Enforcement

Maintainers may moderate, edit, or remove comments and contributions that violate this code, and may restrict participation for repeated or severe violations.

## Reporting

If you observe unacceptable behavior, open a private report to maintainers via GitHub or use the security reporting path in [`SECURITY.md`](SECURITY.md) for sensitive incidents.
