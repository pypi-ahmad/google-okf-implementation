# Support

## Where to Get Help

- Read [`README.md`](README.md) for quick start and command paths.
- Read [`HANDBOOK.md`](HANDBOOK.md) for troubleshooting and deep technical guidance.
- Open an issue for reproducible bugs or feature requests:
  - https://github.com/pypi-ahmad/google-okf-implementation/issues

## What to Include in a Support Request

Please include:
- command(s) you ran
- full error output
- environment details (OS, Python version)
- expected vs actual behavior
- relevant artifact path(s), if any

## Security Issues

For sensitive vulnerabilities, follow [`SECURITY.md`](SECURITY.md) and avoid public disclosure.
